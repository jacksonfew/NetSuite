Using Milestones and Entering Dates

  • Please take your time and work through each milestone.
  • You are all being assigned specific projects- accept and complete each milestone, even if itâ€™s not your name.
  • You are to enter dates (to be explained in this document) on every project
  • Please explore milestones, input data, and become comfortable using them



1. To enter a milestone, simply click the name of the desired milestone


2. Click 'Accept Milestone' and 'Edit.' This will allow you to input the required information and assign the milestone to 'Working'


3. All the required information is always at the bottom of each milestone. You need to input this data before the milestone can be marked complete.


4. You need to save the milestone after any changes made.


5. Understanding the Milestone Schedule Columns

- 'Started On' begins when you accept a milestone (Yellow)

- 'Completed On' is input when you complete a milestone (Green)

- 'Task Status' shows Pending, Working, or Completed (Purple)

- Site Survey Still Shows 'Working' because you have accepted it, but not completed it




6. Completing the Milestone

- to complete a milestone, Make sure the information under the milestone tab is filled in

- Complete by hitting the 'Complete Milestone' Button


7. Adding Dates on Milestones

- You can add any important date from any milestone

- Just click on the header (in this case Estimated Install Date and then 'New Estimated Install Dates' Button


8. Inside a Date Button

- Very Simple, just add the date and click Save


9. Add an estimated install date for practice



Click on 'Estimated Install Dates' within Milestone

- Can be under any milestone, just add another estimated install date
