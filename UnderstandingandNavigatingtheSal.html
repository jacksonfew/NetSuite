Understanding and Navigating the Sales Order Header


  • The above is the full view of the sales order detail (which will be described in following sections)
  • This contains all the general system information before a project (think install case) is created






EDIT BUTTON: You need to click this button in order to modify any field on the sales order


1 – Create Project: Click this button, and it will create the “Install Project”. Once clicked, the project will also be linked under the Project field on the sales order



2 – System Type: Tells you what system you are installing/building


3 – System Prod. Line: Will tell you if it’s a Video/Data/Voice system







4 – Special Install Instructions: If sales has any special notes, it will show up here for the project manager



6 – Special Order Instructions: Think of this as ABP Notes section; same thing. New name. PMs will add notes here specific for the install


7- Project Status: Any change/dates input on milestones will rollup in the project status field for a nice quick view of all pertinent details



8- ABP information: No difference to old systems, tells all the build information status from production/assembly/shipping




























Overall Project Comments: When you add project comments (which you’ll learn in a section to follow) they will rollup here for a summary view. Also, these will be passed back to salesforce for the sales and service team























9 – Drop Files Here: Any attachments you want to add to the sales order/project, just drag and drop here and it will attach it! Also, it will attach it to the salesforce project for sales/support 



Summary: This is the total invoice amount that finance will be invoicing the customer










5 – Customer Profile: Sales now will tell whether an installation is high profile etc… This will auto populated here. If it’s a normal install, it will be blank.

















1 – Manager’s Case Details: Any associated manager’s cases will be displayed in this area, if there are any

2 – Escalation Case Information: Any associated escalations will be displayed in this area, if there are any   

3 - List of Channels: if video system, your channel list for the system will be populated in this area  

4 – Drop Shipping & Special Orders: Any drop shipping information for parts, or special order requests from PMgmt to fulfillment team will appear here
